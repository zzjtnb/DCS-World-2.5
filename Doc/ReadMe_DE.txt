II.INSTALLATION 

Doppelklicken Sie die Setup.exe um die Installation zu beginnen.

Anmerkung: Sie müssen sich mit Administrator-Rechten in Windows einloggen, um das Spiel zu installieren.

Es wird empfohlen Anti-Virus- und Sicherheitssoftware abzuschalten, weil sie den Installationsprozess beträchtlich verlangsamen kann.

Die .MIZ Datei kann Bilder des Flugplan-Briefings sowie Sounddateien und ABRIS Datenbankdateien enthalten.

VIII.TIPPS FÜR BESTE SPIELPERFORMANCE


Wenn Sie eine niedrige Bildwiederholungsrate (FPS) feststellen (die Simulation „ruckelt“), sollten Sie im Optionsmenü folgende Einstellungen anpassen:

• Prüfen Sie, dass die WASSER Darstellung auf „normal“ eingestellt ist

• HITZEFLIMMERN abstellen

• SICHTWEITE vermindern

• SCHATTEN vermindern

• TEXTUREN vermindern

• SZENERIE vermindern

• Auflösung des Spiels vermindern

• MSAA reduzieren (Kantenglättung)

• TSAA reduzieren (transparente Kantenglättung)


ACHTUNG:
Sollten Sie 2 GB oder weniger Arbeitsspeicher installiert haben, so sollten Sie die Grafikeinstellungen generell auf NIEDRIG setzen, um Spielabstürze in großen Missionen zu vermeiden.

TIPS UND TRICKS

Sollten Sie bei einem 32bit Betriebssystem Spielabstürze erleben, so können Sie folgenden Einstellungen ausprobieren:

• Öffnen Sie die DOS Konsole indem Sie erst [WINDOWSTASTE + R] und danach cmd und ENTER drücken

• Geben Sie folgendes ein: bcdedit /set increaseuserva 3072 und drücken ENTER

• Starten Sie Ihren PC neu und überprüfen ob die Abstürze der Vergangenheit angehören

Es kann sein dass Sie temporär das UAC abschalten müssen damit dieser Trick funktioniert.

Einschalten des 3GB-„Schalters“ unter Windows XP:

• Rechts klicken Sie auf den Arbeitsplatz und klicken dann auf „Eigenschaften“.

• Öffnen Sie nun in den Systemeigenschaften auf „Erweitert“.

• Klicken Sie nun auf Einstellungen unter „Starten und Wiederherstellen“.

• Im Fenster „Starten und Wiederherstellen“ finden Sie unter „Systemstart“ die Schaltfläche „Bearbeiten“. Klicken Sie bitte darauf. Die Datei „boot.ini“ Ihres Windows-Betriebssystems wird dadurch im Microsoft® Notepad geöffnet.

• Erstellen Sie zunächst eine Sicherheitskopie Ihrer „boot.ini“-Datei. Da sich diese Datei von Computer zu Computer unterscheidet, können Sie sie nicht von einem anderen Computer wiederherstellen.

• Markieren Sie die folgende Zeile in Ihrer „boot.ini“-Datei:
multi(0)disk(0)rdisk(0)partition(2)\WINDOWS="Micro soft Windows XP Professional" /fastdetect

• Drücken Sie STRG + C, um die Zeile zu kopieren. Anschließend drücken Sie STRG + V, um die Zeile direkt unter die Original-Zeile einzufügen.

Bitte beachten Sie: Die Zeile in Ihrer Datei kann sich von der hier beispielhaft dargestellten Zeile in wichtigen Details unterschieden. Stellen Sie daher sicher, dass Sie für das Kopieren nur Ihre eigene „Boot.ini“-Zeile benutzen, nicht etwa die hier stehenden!

• Modifizieren Sie nun bitte die Zeile, in dem Sie folgenden Zusatz einfügen: „/3GB“. Die Zeile sollte anschließend etwa so aussehen:
multi(0)disk(0)rdisk(0)partition(2)\WINDOWS="Micro soft Windows XP Professional 3GB" /3GB /fastdetect

Wichtig: Bitte überschreiben Sie bei dieser Modifikation keine anderen Zeilen in Ihrer „Boot.ini“-Datei!

• Speichern und schließen Sie nun die „boot.ini“-Datei.

• Drücken Sie bei allen offenen Dialogfeldern „OK“.

• Starten Sie Ihren Computer neu.

• Während des Neustarts, wählen Sie die Option mit dem 3GB-Zusatz. Sie laden damit die modifizierte Konfiguration. Wählen Sie die Option nicht, startet Ihr System in der normalen Konfiguration mit der 2GB-Speicher-Einstellung.

Bemerkung: Sollten während des Neustarts Fehlermeldungen auftreten, müssen Sie eventuell einige Treiber auf den neusten Stand bringen.

X.MULTI-DISPLAY SETTINGS

Sie können die Spieldarstellung auf bis zu drei Monitoren darstellen lassen.

Zu diesem Zweck muss Ihr Videotreiber den Desktop-Arbeitsplatz auf mehrere Monitore verteilen können. Im Optionsbildschirm wählen Sie die Bildschirmauflösung und die erforderliche Monitor-Voreinstellung (1 Kamera, 3 Kameras) in den Grafikoptionen aus.

Die Voreinstellungen sind im …\Config\MonitorSetup Ordner zu finden.

XII.FAQ

http://forums.eagle.ru/showthread.php?t=88164

Frage: Warum habe ich auf meinem Windows Vista / Windows 7 System Probleme beim Installieren von DCS World?

Antwort: Starten Sie das Installationsprogramm manuell mit der Setup.exe Datei. Stellen Sie sicher, dass Sie die Setup-Datei mit Administratorrechten starten (Rechtsklick und die entsprechende Option auswählen). Wenn Sie immer noch Probleme haben, deaktivieren Sie die Benutzerkontensteuerung (UAC) beim ersten Start von DCS World.

Frage: Ich benutze die launcher.exe Datei aus dem \...\bin\stable\ Ordner, um das Spiel zu starten, aber es startet nicht oder ergibt eine Fehlermeldung.

Antwort:  Die Simulation sollte nur durch die Desktop-Symbole oder die Startmenü-Einträge gestartet werden. 
 
Frage: Warum kehrt die Simulation zum Desktop zurück, wenn sie eine Mission lädt?

Antwort: Das Benutzerinterface und die Spiel-Engine sind getrennte Elemente und Sie sehen, wie die Simulation von einem zu dem anderen Element umschaltet. Es gibt zwei Gründe dafür: 1) Wird die GUI vom Spiel getrennt, setzt dies Leistungsreserven frei für die Spiel-Engine, und 2) ist die GUI unter Verwendung neuerer Technologie geschrieben, die noch nicht rückwärts-kompatibel mit der Spiel-Engine ist. 

Frage: Warum ist die Trimmung auf dem Microsoft Force Feedback 2 Joystick invertiert?

Antwort: Dies ist eine Eigenheit des Gerätes. Die normale Achsenfunktionalität kann im Optionsmenü auf dem Reiter Steuerung wiederhergestellt werden, und zwar durch Aktivieren des "Achsen vertauschen" Kontrollkästchens im ForceFeedback Einstellungsdialog. 

Frage: Das Spiel reagiert nicht auf Tastaturbefehle und ich kann den Pausemodus nicht beenden bzw. das Spiel nicht starten. Meine Joystick-Profile lassen sich nicht speichern oder wiederherstellen.

Antwort: Wenn Sie das Programm „NewView“ benutzen, um das Spiel zu starten, ist dies wahrscheinlich die Ursache für das Problem. Sie müssen entweder mit ALT-TAB aus dem Spiel zum Desktop und wieder zurück ins Spiel wechseln, oder den NewView Ordner vor dem Starten der Simulation umbenennen. 

Frage: Ich habe die nächste Mission in einer Kampagne geladen, hab mich dann aber umentschieden und habe die Simulation ins Hauptmenü verlassen, ohne das Spiel zu starten (Pause Modus an). Nun bekomme ich die Meldung dass ich die vorherige Mission nochmals fliegen soll. Was sollte ich tun?

Antwort: Es ist ein Feature des Kampagne. Wenn Sie eine Kampagnenmission verlassen möchten ohne diese zu fliegen, so sollten Sie die Simulation für mindestens eine Sekunde starten, und dann das Spiel verlassen.
